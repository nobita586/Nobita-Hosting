## 🛠️ Installation

### Quick Start

```bash
# Download the main menu script
curl -o nobita-menu.sh https://raw.githubusercontent.com/nobita586/Nobita-Hosting/main/menu.sh

# Make it executable
chmod +x nobita-menu.sh

# Run the menu
sudo ./nobita-menu.sh
